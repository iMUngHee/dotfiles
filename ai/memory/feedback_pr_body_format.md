---
name: PR body formatting
description: Do not use numbered sub-headings in PR bodies. Use flat bullet lists instead.
type: feedback
---

When writing or editing PR body content:
- Do NOT use numbered sub-headings (`### 1. ...`, `### 2. ...`)
- Use flat bullet points (`- ...`) to separate items under `## 변경 내용`
- Match the user's existing format exactly — do not restructure or reformat
